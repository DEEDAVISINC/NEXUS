## **üìä TEMPLATE VARIABLES REFERENCE**

### **Common Variables (Used in Multiple Templates):**
- `{{clientName}}` - Client's full legal name
- `{{date}}` - Current date (auto-filled)
- `{{claimNumber}}` - Internal reference number (LBPC-####)
- `{{property}}` - Property street address
- `{{city}}` - Property city
- `{{state}}` - Property state (2-letter code)
- `{{zipCode}}` - Property zip code
- `{{county}}` - County name
- `{{surplusAmount}}` - Total surplus funds available

### **Financial Variables:**
- `{{totalClaimAmount}}` - Total claim amount
- `{{firmFeeAmount}}` - 30% fee (auto-calculated)
- `{{clientPaymentAmount}}` - 70% to client (auto-calculated)
- `{{yourFeeAmount}}` - Same as firmFeeAmount
- `{{clientPortion}}` - Same as clientPaymentAmount

### **Contact Variables:**
- `{{clientPhone}}` - Client phone number
- `{{clientEmail}}` - Client email address

### **Legal Document Variables:**
- `{{companyRepName}}` - LBPC representative name
- `{{companyRepTitle}}` - LBPC rep title (e.g., "Claims Manager")
- `{{countyName}}` - County holding funds
- `{{countyTreasurerName}}` - County treasurer name
- `{{municipalityName}}` - Municipality name
- `{{documentDescription}}` - Description of certified document
- `{{documentHolderName}}` - Who possesses the document
- `{{notaryName}}` - Notary public name
- `{{day}}`, `{{month}}`, `{{year}}` - Notary date components

### **Agent Variables:**
- `{{agentName}}` - Agent/rep name (usually "Charin Cross")
