## ğŸ—“ï¸ CALENDAR LINK GENERATOR:

**Google Calendar Link Format:**
```
https://calendar.google.com/calendar/render?action=TEMPLATE&text=[EVENT_TITLE]&dates=[START_DATE]/[END_DATE]&details=[DESCRIPTION]&sf=true&output=xml
```

**How to create:**
1. **Event Title:** `Quote+Due:+[YOUR_BID_NAME]`
2. **Start Date:** `YYYYMMDDTHHMMSSZ` (UTC time)
3. **End Date:** Same as start + 1 hour
4. **Description:** `Submit+quote+to+Dee+Davis`

**Example:**
```
https://calendar.google.com/calendar/render?action=TEMPLATE&text=Quote+Due:+Madison+Heights+Lawn&dates=20260124T220000Z/20260124T230000Z&details=Submit+quote+to+Dee+Davis+-+Madison+Heights+lawn+contract
```

**Quick Converter:**
- Friday, Jan 24, 2026 @ 5pm EST = `20260124T220000Z` (UTC is EST+5)
- Monday, Jan 27, 2026 @ 4pm CST = `20260127T220000Z` (UTC is CST+6)
